ZoneOffset zoneOffset()

Obtains the current time zone offset.

Implementation

ZoneOffset zoneOffset() {
    var dateTime = DateTime.now();
    var zoneOffset = dateTime.timeZoneOffset;
    var offset = ZoneOffset();
    offset.amountSeconds = zoneOffset.inSeconds;
    return offset;
}
