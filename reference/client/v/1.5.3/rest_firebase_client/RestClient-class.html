RestClient class

A FirebaseClient based on the Firebase REST API.

This implementation does not have platform limitations.

See WebFirebaseClient for a web-specific implementation.

Note: the childAdded, childChanged and childRemoved event streams are not supported by this client implementation. It thus does not support Spine subscriptions. If you need the subscriptions functionality, consider switching to the Firebase Admin SDK based client.
